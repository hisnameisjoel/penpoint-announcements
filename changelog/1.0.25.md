::party-popper:: **New in v1.0.25**

- **Relationship Block Swap Roles** — Two-way relationships with different labels can now display in either order. By default, each name shows the role they play *for* the current character. Flip on "Swap Role Labels" in the Relationships block settings to pair each name with their own role instead.

::bug:: **Bug Fixes**

- **Onboarding on Small Screens** — The new-user walkthrough now scales properly on smaller screens and at larger font sizes, from compact laptops to 4K displays.
- **Profile Picture Cropping** — The crop dialog for character, item, and location images is fully fixed — drag, zoom, and buttons all work correctly. The crop window now adapts to portrait and landscape images instead of a fixed square.
- **Searchbar Overlapping Toolbar** — At narrower window widths, the search bar no longer drifts into the toolbar buttons. It now slides left to keep everything visible.

Thank you for using Penpoint!
