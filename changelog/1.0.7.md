# What's New in Penpoint 1.0.7

## üéâ New Features

### Smooth Page Transitions
Pages now transition smoothly when navigating between Characters, Items, Locations, Chapters, and Notes. Content is fully loaded before fading in, giving you a polished, app-like experience. Notes get extra flair with zoom-in/out animations!

### Relationship Map Customization
Personalize your relationship map with:
- **Adjustable line arc strength** - Control how curved your relationship lines appear
- **Toggleable labels** - Show or hide relationship labels for a cleaner view
- **Three color saturation modes** - Choose between Grayscale, Normal, or Full color intensity
- **Improved hover behavior** - Better visual feedback in the detail panel

### Items View Modes
Choose how you want to view your items:
- **Cards** - Multi-column grid layout for visual browsing
- **Compact** - Dense list for maximum information density
- **Standard** - Comfortable list with balanced spacing

Access via the Settings gear icon. Your preference persists across sessions.

### Global Create Shortcut (Ctrl+N)
Quickly create new items on any page with **Ctrl+N** (Cmd+N on Mac). No more hunting for the "Add" button!

### What's New Button
Review the complete changelog for your current version anytime from **Settings ‚Üí About**. Never miss what's changed.

### Smooth Theme Transitions
Theme colors now fade smoothly when switching between light and dark themes. No more jarring color jumps!

## üíÖ Improvements

### Enhanced Tag Color Palette
The tag color palette has been expanded from 8 to 12 vibrant colors with 50%+ increased saturation for better visual distinction. Your tags will pop!

### Locations Expand/Collapse Animations
Added **Expand All / Collapse All** button with smooth slide animations. Managing nested location hierarchies is now a breeze.

### Detail Panel Improvements
- Consistent Edit/View toggles across all panels
- Removed redundant close buttons for cleaner interface
- Improved Items panel space allocation (63%/37% split)

### "Add New" Buttons
"Add New" buttons now appear as the first list item for easier access without scrolling to the bottom.

### Smarter Character Creation
Character creation now defaults to the importance level of your most recently created character, saving you repetitive clicks.

## üêõ Bug Fixes

### Rich Text Fixes
- **Multiple blank lines** - Blank lines are now preserved correctly in all rich text fields (bios, descriptions, notes, chapter details)

### @ Mention System Overhaul
Fixed numerous issues with the @ mention system:
- Search failures when typing @ symbol
- Insertion issues that would sometimes fail
- Dropdown positioning now sits directly above cursor
- Dropdown scales properly with font size
- Clipping issues in modals
- Esc key now properly closes the mention dropdown

### Navigation & UI Fixes
- Fixed back button skipping over "empty" page states (all tags, all locations, all items)
- Fixed trait block dropdowns (like Alignment) causing page to jump and lose scroll position
- Fixed Ctrl+E shortcut not working on Items and Locations pages
- Fixed searchbar shifting horizontally between pages
- Fixed items jumping without animation when changing sort/filters

### Data & Import Fixes
- Fixed book ID mismatch warning when importing backup files (.penpoint)
- Fixed chapter numbers not updating after adding/deleting chapters
- Fixed Locations dropdown in character sheets not allowing typing
- Fixed location descriptions not updating when switching in edit mode

### Theme & Visual Fixes
- Fixed checkbox checkmarks not adapting to theme colors
- Fixed dual column layout pushing add form to the right when empty
- Improved page transition performance
