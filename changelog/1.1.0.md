# Penpoint v1.1.0

::party-popper:: **New Features**

- **Right-Click Menus on Chapters, Items & Locations** — Right-click chapter cards to open, jump to Manuscript, reorder, insert, or delete. Right-click items or locations to add, delete, change importance, sort, and more. Detail panels have their own right-click menus too — edit mode, profile image, type changes, all without hunting through the UI.
- **Writing Stats** — Your analytics dropdown now shows your current manuscript word count with today's change, plus a quick look at your last writing day — great for picking up after a break.
- **Sort Locations** — The Locations sort menu now includes Ascending and Descending options for any sort method.

::bug:: **Bug Fixes**

- **Opening a Chapter in Manuscript** — Fixed the first click loading the wrong chapter. It goes to the right one every time now.
- **Multi-Value Traits on Import** — Importing characters with multiple trait values no longer collapses them into one.
- **Typing Glitch on Mac** — Fixed cursor jumping to the start of a sentence while typing on some Macs.
- **Character List Full Screen** — Expanding the character list now fills the width with as many columns as will fit.
- **Deleted Items Staying on Screen** — The detail panel now closes cleanly when an item is deleted.
- **Spellcheck Possessives** — Writing "Charlye's" no longer gets flagged when "Charlye" is whitelisted.

Thank you for using Penpoint!
